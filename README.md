# ♻️ EcoPonto

Aplicação web **mobile-first** desenvolvida em **React + TypeScript** que permite localizar pontos de coleta de materiais recicláveis próximos ao usuário.

O objetivo do projeto é conectar pessoas que desejam descartar resíduos recicláveis com empresas que realizam coleta e reciclagem.

---

## 🎯 Objetivo do projeto

Facilitar o descarte correto de materiais recicláveis por meio de uma plataforma que permite:

- Localizar pontos de coleta próximos
- Visualizar horários de funcionamento
- Acessar rotas até o local
- Filtrar pontos por tipo de material

---

## ⚙️ Funcionalidades

### 👤 Usuário

- Localizar pontos de coleta próximos
- Filtrar por tipo de material
- Visualizar horáro de funcionamento
- Traçar rota até o local

### 🏢 Empresas

- Cadastro de conta
- Login
- Cadastro de ponto de coleta
- Definifição de tipos de materiais aceitos
- Definição de horário e dias de funcionamento

## 🚀 Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/valerianocarolina/ecoponto-app.git
```

Entre na pasta:

```bash
cd ecoponto-app
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm start
```
